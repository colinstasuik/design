In all my work, I attempt to build solutions that play with simple forms in unexpected ways. I think that a lot can be represented with basic shapes and solid typographic principles. I enjoy both leaning on these principles and breaking them when it feels like a more visually-interesting solution.

I am currently searching for a full-time role where I can put these ideas to work. My ideal spot would be within a creative studio that gets excited about the same things I do, or with a company that believes good design is necessary for overall success. Some specific industries I am heavily interested in include robotics, AI, neuroscience, fashion, athletics, and animal/planet conservation.

I have always considered myself a graphic designer, but am working on expanding my skillset with the goal of being able to justifiably call myself a creative technologist. This website serves as one project in service of that goal, being the first that I have coded from scratch. It was built in Visual Studio Code with HT​ML, CSS, and Javascript, and I'm now learning Tailwind and React to improve it. I am very focused on building on these basic development skills and introducing motion capabilities to my skillset as well.
